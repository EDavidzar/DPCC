# DPCC
Data Protection Control Center

1. INTRODUCCIÓN
El programa ayuda las empresas a cumplir sus obligaciones con respecto a la protección de datos personales.
Se utiliza un servidor de base de datos MYSQL/MARIADB en los que se introducen los datos necesarios para el cumplimiento de la LOPD y se pueden crear varios documentos y anexos necesarios.
2. LENGUAJE DE PROGRAMACIÓN Y ENTORNO OPERATIVO
Ha sido desarrollado en el lenguaje de programación Java y se puede ejecutar en cualquier sistema/entorno operativo en el que se pueda instalar, esté instalado y configurado  Java Runtime Environment (JRE). versión mínima 17.0 de Java Runtime Environment (JRE).

1. INTRODUCTION
The program helps companies comply with their obligations regarding personal data protection.
It uses a MYSQL/MARIADB database server into which the data necessary for compliance with the LOPD (Spanish Data Protection Law) is entered, and various necessary documents and annexes can be created.
2. PROGRAMMING LANGUAGE AND OPERATING ENVIRONMENT
It has been developed in the Java programming language and can be run on any operating system/environment on which Java Runtime Environment (JRE) can be installed, is installed, and is configured. Minimum version 17.0 of Java Runtime Environment (JRE)
